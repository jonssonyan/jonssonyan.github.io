安装 MySQL

系统使用的是 MySQL 数据库，所以要在 Docker 中安装 MySQL

参数解释：

  • -d: 后台运行容器，并返回容器 ID
  • –name: 为容器指定一个名称
  • -p: 指定端口映射，格式为：主机(宿主)端口:容器端口
  • -v: 绑定一个卷，容器的 /var/lib/mysql 映射到 主机的目录 /data/mysql
  • -e MYSQL_ROOT_PASSWORD=123456: 设置环境变量，密码设置为 123456
  • mysql:5.7.31：使用镜像 mysql:5.7.31
