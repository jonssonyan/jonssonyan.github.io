不能直接使用 Arrays.asList 来转换基本类型数组

代码

执行结果

原因是asList 方法的参数必须是对象或者对象数组，而原生数据类型不是对象，当传入一个原生数据类型数组时，asList 的真正得到的参数就不是数组中的元素，而是数组对象本身。

解决方案
